Features

Everything you need for NFC

A complete toolkit built with Jetpack Compose and Material Design 3.

Read Tags

Read Mifare Classic 1K/4K tags sector by sector with automatic key detection and extended dictionary support.

Write Magic Cards

Write complete dumps to CUID/Gen1a magic cards with proper authentication and block-by-block verification.

HCE Emulation

Emulate NFC tags via Host Card Emulation. Full emulation available with root + NXP chipset.

Hex Editor

View and edit raw hex data with color-coded keys, access bits, and sector trailer highlighting.

Key Cracking

Crack unknown keys using 9+ built-in dictionaries including Flipper Zero, Proxmark, and VIGIK databases.

Encrypted Storage

All saved tags are encrypted with AES-256. Export and import encrypted backups with one tap.

How it works

Three simple steps

From physical tag to digital clone in seconds.

01

Scan

Hold your phone near a Mifare Classic tag. The app reads all sectors and cracks keys automatically.

02

Save

The complete dump is saved and encrypted locally. View hex data, keys, and access bits in the editor.

03

Clone

Write the dump to a magic card or emulate it directly via HCE. Your tag is now digital.
